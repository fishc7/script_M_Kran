# 📋 Проект M_Kran - Полное описание

## 🎯 Обзор проекта

**M_Kran** - это комплексная система управления данными сварочных работ для промышленных проектов. Система предназначена для обработки, анализа и контроля качества сварочных соединений в рамках крупных строительных проектов.

### 📊 Основные характеристики

- **Тип системы**: Веб-приложение с модульной архитектурой
- **База данных**: SQLite с 32 таблицами
- **Размер проекта**: 383.22 МБ
- **Количество записей**: 100,000+ записей сварочных работ
- **Языки**: Python (Flask), JavaScript, HTML/CSS
- **Архитектура**: Модульная, масштабируемая

## 🔌 API система

### 📡 Основные API эндпоинты

#### 📊 Статистика и данные
- `GET /api/title_stats` - Статистика по титулам
- `GET /api/update_stats` - Обновление статистики
- `GET /api/status_statistics` - Статистика статусов
- `GET /api/results_statistics` - Статистика результатов

#### 🔍 Поиск и фильтрация
- `GET /api/table/<name>/search` - Поиск в таблице
- `GET /api/table_info` - Информация о таблицах
- `GET /api/get_tables` - Список таблиц
- `POST /api/get_columns` - Колонки таблицы

#### 📁 Работа с файлами
- `GET /export/<table>` - Экспорт таблицы в Excel
- `GET /export_filtered/<table>` - Экспорт с фильтрацией
- `GET /download/<file>` - Скачивание файлов
- `POST /delete_file/<file>` - Удаление файлов

#### 🔧 Управление скриптами
- `POST /run_script` - Запуск скрипта
- `POST /run_etl_stage` - Запуск ETL этапа
- `POST /run_etl_pipeline` - Запуск ETL пайплайна
- `GET /script_status/<id>` - Статус выполнения

#### 🛠️ Системные функции
- `POST /api/backup` - Создание бэкапа
- `GET /api/backups/list` - Список бэкапов
- `POST /api/backups/restore` - Восстановление
- `POST /api/shutdown` - Завершение работы
