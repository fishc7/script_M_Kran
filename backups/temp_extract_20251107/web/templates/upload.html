Информация о загрузке
Поддерживаемые форматы:
  • Excel файлы (.xlsx, .xls) - для загрузки данных
  • CSV файлы (.csv) - для импорта данных
  • PDF файлы (.pdf) - для обработки документов
Ограничения:
  • Максимальный размер файла: 16 МБ
  • Файлы сохраняются в папке uploads
  • Оригинальные имена файлов сохраняются (включая русские символы)
  • При совпадении имен добавляется номер в скобках
Совет: После загрузки файла используйте соответствующие скрипты для его обработки.
