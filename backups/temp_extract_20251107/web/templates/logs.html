Экспорт данных

Выгрузка данных в различных форматах для отчетности

Экспорт
Информация о журналах
Журнал НГС Эксперт

Основной источник данных о контроле качества. Содержит информацию о ВИК (визуальный контроль), РК (радиографический контроль) и других видах контроля.

Журнал сварки

Детальная информация о сварочных работах, включая параметры сварки, квалификацию сварщиков и результаты контроля сварных швов.

Журнал ремонта

Данные о ремонтных работах, устранении дефектов и восстановлении качества продукции после выявления несоответствий.

Журнал результатов ПОСЛЕ Ремонта

Отчеты по неразрушающему контролю ПОСЛЕ Ремонта с результатами испытаний и сертификатами качества продукции.

Последнее состояние стыка

Аналитическая страница для мониторинга актуального состояния сварных соединений с радиографическим контролем (RT). Предоставляет автоматический выбор последних записей, подсчет количества RT записей, динамическую статистику по статусам РК, продвинутую фильтрацию и экспорт данных в Excel.

Статистика по статусам

Детальная аналитика по статусам РК и ВИК с визуализацией данных. Показывает распределение статусов, процентные соотношения, количество пустых записей и предоставляет интерактивные карточки статистики.
