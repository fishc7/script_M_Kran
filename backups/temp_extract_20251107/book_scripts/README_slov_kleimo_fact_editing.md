# Руководство по редактированию поля ФИО в таблице слов_клейм_факт

## Описание
Добавлена возможность редактирования поля ФИО в таблице слов_клейм_факт через веб-интерфейс.

## Как использовать

### 1. Доступ к таблице
- Откройте веб-интерфейс M_Kran
- Перейдите в раздел "База данных"
- Выберите таблицу "Словарь клейм факт" (слов_клейм_факт)

### 2. Редактирование поля ФИО
- Найдите запись, которую хотите отредактировать
- **Двойной клик** по полю ФИО в нужной строке
- Откроется модальное окно редактирования
- Введите новое значение ФИО
- Нажмите кнопку "Сохранить"

### 3. Структура таблицы
Таблица слов_клейм_факт содержит следующие поля:
- **id** - уникальный идентификатор записи
- **Фактическое_Клеймо** - клеймо сварщика (только для чтения)
- **ФИО** - фамилия, имя, отчество (редактируемое поле)
- **Примечание** - дополнительная информация (только для чтения)
- **created_at** - дата создания записи
- **updated_at** - дата последнего обновления

### 4. Особенности
- Только поле ФИО можно редактировать
- Остальные поля доступны только для просмотра
- При сохранении автоматически обновляется поле updated_at
- После сохранения страница автоматически перезагружается

## Технические детали

### API Endpoints
- `GET /api/slov_kleimo_fact/<id>` - получение записи
- `PUT /api/slov_kleimo_fact/<id>` - обновление записи

### JavaScript функции
- `openSlovKleimoFactEditModal()` - открытие модального окна
- `showSlovKleimoFactEditModal()` - отображение формы редактирования
- `saveSlovKleimoFactChanges()` - сохранение изменений

- Проверьте подключение к базе данных
