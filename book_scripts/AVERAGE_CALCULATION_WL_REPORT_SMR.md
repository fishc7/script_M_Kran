# Расчет средних значений в таблице wl_report_smr

## Описание

Данный документ описывает методы расчета средних значений в таблице `wl_report_smr` базы данных SQLite.

## Структура таблицы

Таблица `wl_report_smr` содержит данные об отчетах мастеров строительно-монтажных работ (СМР) по сварке.

### Основные поля:
- **Дата_сварки** (или `Date_welding`) - дата выполнения сварочных работ (формат: YYYY-MM-DD)
- **КЛЕЙМО** - клеймо сварщика
- **Факт** - фактическое значение
- **КЛЕЙМО_1**, **Ревизия**, **Unnamed:_16** - дополнительные поля с клеймами
- **Фактическое_Клеймо** - итоговое клеймо (приоритетное значение)
- **ЛИНИЯ** - номер линии
- **ISO** - ISO стандарт
- **СОЕДИНЕНИЕ_** - информация о соединении
- **Деталь1**, **Деталь2** - детали соединения
- **date_load** - дата загрузки данных

## Методы расчета средних значений

### 1. Среднее значение на каждый день

Для расчета среднего значения на каждый день необходимо:
1. Группировать данные по полю `Дата_сварки`
2. Применить агрегатную функцию `AVG()` к числовому полю
3. Отфильтровать NULL значения

#### SQL запрос для расчета среднего на каждый день:

```sql
-- Пример 1: Среднее значение по дате (если есть числовое поле)
SELECT 
    Дата_сварки,
    COUNT(*) as количество_записей,
    -- Замените 'числовое_поле' на реальное числовое поле из таблицы
    AVG(CAST(числовое_поле AS REAL)) as среднее_значение
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
    AND числовое_поле IS NOT NULL
GROUP BY Дата_сварки
ORDER BY Дата_сварки;
```

#### Пример с использованием Python (pandas):

```python
import sqlite3
import pandas as pd

# Подключение к базе данных
db_path = "путь/к/базе/данных/M_Kran_Kingesepp.db"
conn = sqlite3.connect(db_path)

# Чтение данных
query = """
SELECT 
    Дата_сварки,
    числовое_поле
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
    AND числовое_поле IS NOT NULL
"""

df = pd.read_sql_query(query, conn)

# Группировка по дате и расчет среднего
daily_average = df.groupby('Дата_сварки')['числовое_поле'].mean().reset_index()
daily_average.columns = ['Дата', 'Среднее_значение']

print(daily_average)

conn.close()
```

### 2. Среднее значение на весь период

Для расчета среднего значения на весь период:
1. Применить агрегатную функцию `AVG()` ко всем записям
2. Отфильтровать NULL значения

#### SQL запрос для расчета среднего на весь период:

```sql
-- Среднее значение на весь период
SELECT 
    COUNT(*) as общее_количество_записей,
    AVG(CAST(числовое_поле AS REAL)) as среднее_значение_за_период,
    MIN(Дата_сварки) as первая_дата,
    MAX(Дата_сварки) as последняя_дата
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
    AND числовое_поле IS NOT NULL;
```

#### Пример с использованием Python (pandas):

```python
import sqlite3
import pandas as pd

# Подключение к базе данных
db_path = "путь/к/базе/данных/M_Kran_Kingesepp.db"
conn = sqlite3.connect(db_path)

# Чтение данных
query = """
SELECT 
    Дата_сварки,
    числовое_поле
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
    AND числовое_поле IS NOT NULL
"""

df = pd.read_sql_query(query, conn)

# Расчет среднего на весь период
period_average = df['числовое_поле'].mean()
total_count = len(df)
first_date = df['Дата_сварки'].min()
last_date = df['Дата_сварки'].max()

print(f"Общее количество записей: {total_count}")
print(f"Среднее значение за весь период: {period_average:.2f}")
print(f"Период: с {first_date} по {last_date}")

conn.close()
```

## Важные замечания

### 1. Определение числовых полей

Перед расчетом средних значений необходимо определить, какие поля в таблице являются числовыми. Для этого можно использовать:

```sql
-- Просмотр структуры таблицы
PRAGMA table_info(wl_report_smr);
```

### 2. Обработка NULL значений

- Всегда фильтруйте NULL значения перед расчетом среднего
- Используйте `WHERE числовое_поле IS NOT NULL` в SQL запросах
- В pandas используйте `dropna()` или `notna()` для фильтрации

### 3. Преобразование типов данных

Если числовые данные хранятся как TEXT, необходимо преобразовать их:

```sql
-- Преобразование TEXT в REAL для расчета среднего
AVG(CAST(числовое_поле AS REAL))
```

### 4. Группировка по дате

При группировке по дате убедитесь, что формат даты единообразен (YYYY-MM-DD).

## Примеры использования

### Пример 1: Среднее количество записей на каждый день

```sql
SELECT 
    Дата_сварки,
    COUNT(*) as количество_записей
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
GROUP BY Дата_сварки
ORDER BY Дата_сварки;
```

### Пример 2: Среднее количество записей на весь период

```sql
SELECT 
    COUNT(*) as общее_количество,
    COUNT(DISTINCT Дата_сварки) as количество_дней,
    CAST(COUNT(*) AS REAL) / COUNT(DISTINCT Дата_сварки) as среднее_на_день
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL;
```

### Пример 3: Статистика по датам с дополнительной информацией

```sql
SELECT 
    Дата_сварки,
    COUNT(*) as количество_записей,
    COUNT(DISTINCT КЛЕЙМО) as количество_уникальных_клейм,
    COUNT(DISTINCT ЛИНИЯ) as количество_линий
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
GROUP BY Дата_сварки
ORDER BY Дата_сварки;
```

## Дополнительные агрегатные функции

Помимо среднего значения, можно рассчитать:

- **MIN()** - минимальное значение
- **MAX()** - максимальное значение
- **SUM()** - сумма значений
- **COUNT()** - количество записей
- **STDDEV()** - стандартное отклонение (если доступно)

Пример:

```sql
SELECT 
    Дата_сварки,
    COUNT(*) as количество,
    MIN(числовое_поле) as минимум,
    MAX(числовое_поле) as максимум,
    AVG(CAST(числовое_поле AS REAL)) as среднее,
    SUM(CAST(числовое_поле AS REAL)) as сумма
FROM wl_report_smr
WHERE Дата_сварки IS NOT NULL
    AND числовое_поле IS NOT NULL
GROUP BY Дата_сварки
ORDER BY Дата_сварки;
```
